## ðŸš¨ Common Issues & Fixes

### Website not loading after Netlify deploy
- Check build settings (should be empty)
- Verify publish directory is `/`
- Check Netlify deploy log for errors

### Domain not connecting
- DNS can take 24-48 hours to propagate
- Clear browser cache
- Check DNS records match Netlify's

### Tally forms not receiving submissions
- Check email notifications enabled
- Verify form URL is correct in website
- Test form submission yourself

### WhatsApp not sending messages
- Check internet connection
- Verify phone number format (+1234567890)
- Restart WhatsApp app
