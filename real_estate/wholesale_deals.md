## What We Need (Wholesale Criteria)

### Good for Fix & Flip
- [ ] Structure intact (roof, walls ok)
- [ ] Cosmetically distressed (ugly but fixable)
- [ ] Vacant
- [ ] Price: $10K-$50K
- [ ] ARV (after repair value): $80K+

### Red Flags
- [ ] Major structural issues
- [ ] Occupied (can't get possession)
- [ ] Federal liens (can't clear)
- [ ] Foundation/structural damage

## Due Diligence for Wholesale

### Step 1: Property Info
- [ ] Get parcel number
- [ ] Check OPA value (market value)
- [ ] Check last sale price
- [ ] Check year built

### Step 2: Condition
- [ ] Drive-by (or hire runner)
- [ ] Photos of exterior
- [ ] Note: roof, windows, doors, yard
- [ ] Check: occupied or vacant?

### Step 3: Title
- [ ] Check liens (county website)
- [ ] Estimate: taxes owed + fees
- [ ] Calculate: MAX offer = ARV - repair - fees - margin

### Step 4: Numbers
```
ARV (After Repair Value): $_______

- Selling Costs:         -$_______

```
